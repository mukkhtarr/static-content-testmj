Intro to this section.

Power source

This section is for your Power Source. Your current selection(s) are repeated on the righthand, “proposed” side of the screen. Make any changes to the righthand side of the screen by selecting or deselecting the appropriate checkboxes. Select Continue Application to move on to the next page.

Water supply source

This section identifies your Source of Water Supply. This is the same information recorded on your current license. This section can be used to edit, add, and delete rows. To edit a row, select the checkbox next to one of your sources and select Edit Selected. Additionally, you can select Actions and Edit from the row you would like to change.

A new window will open. Select the Type of Water Supply from the first drop down list.

  • For Retail Supplier the following fields are required: name of supplier.
  • For Groundwater Well the following fields are required: geographic location coordinates, and maximum amount of water diverted.
  • For Rainwater Catchment System the following fields are required: geographic location coordinates, total square footage, total storage capacity, and description.
  • For Small Retail Water Supplier where the source of water is a diversion, the following fields are required: name of supplier, geographic location coordinates, authorized place of use, maximum amount of water delivered in gallons to the commercial cannabis business in any year, and water source. For Small Retail Water Supplier where the source of water is a well, the following fields are required: name of supplier, geographic location coordinates, and the maximum amount of water delivered in gallons to the commercial cannabis business in any year.
  • Status field must be updated with one of the following statuses: Delete, Modify, or New.

To add a new source of water supply, select Add a Row. You can add up to 10 Rows at a time.

A new window will open, allowing you to add required information. Refer to the previous section to complete the required fields for each new source. The Status field will automatically select New, and the New Status box will be checked.
When you’ve made any required changes, select Submit to close the form.
Select Continue Application to move on to the next page.

Electricity usage

This section identifies your Electricity Usage and Average Weighted GGEI. Enter the Electricity Usage information for the previous licensed period. To begin, click Add a Row.

If a change in a prior renewal year is needed, select the checkbox next to the appropriate reporting year and select Edit Selected or select Actions and Edit.
If reporting for a previous year that has not been recorded, select Add a Row.

A new window will open. Select the Reporting Year then select the Usage Type.

For Utility, the following fields are required: name of utility provider, total electricity supplied, and the total electricity supplied by zero net energy renewable sources (if applicable), and the GGEI.
For Off Grid Renewable Resource, the following fields are required: type of off grid renewable source, and the total electricity supplied by zero net energy renewable sources.
For Other Source, the following fields are required: type of other source, total electricity supplied, and GGEI.
Select Submit once complete.

Average weighted GGEI

Enter the Average Weighted GGEI information for the previous licensed period. To begin, click Add a Row. If a change in a prior renewal year is needed, select the checkbox next to the appropriate reporting year and select Edit Selected or select Actions and Edit. If reporting for a previous year that has not been recorded, select Add a Row. A new window will open.

Select the Reporting Year. Enter the Average Weighted GGEI.
If this is a new entry, in the Status field select New. If you are modifying an existing entry, select Modify.
Select Submit once complete.

Note: To make changes to existing electricity usage information the status field must be updated with Modify. If a row requires to be deleted, do not select the row status as Delete. Update the status field as Modify and upload a written statement requesting the removal with any supporting documents.
