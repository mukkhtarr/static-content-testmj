You will need to input electricity usage data through the licensing portal in order to renew your license. Review the different types of data you will need to provide and resources to help you with this step.

Electricity Usage Data Tables

Provide the required electricity usage for each power source during the previous annual licensed period, also referred to as the Reporting Year.

For each power source used for commercial cannabis cultivation activities, select “Add a Row” to open a new window.

Select the appropriate “Reporting Year” from the dropdown list. Select a new “Usage Type” in the dropdown list, this will reconfigure the other fields to display only the values required for that usage type.

Any field with an asterisk is a required field and must be completed to proceed. Select “Submit” once complete. 

Average Weighted GGEI Data Tables

Select “Add a Row” to open a new window. Select the appropriate “Reporting Year” from the dropdown list.

Any field with an asterisk is a required field and must be completed to proceed. Select “Submit” once complete.

For assistance in compiling this information see the Electricity Usage Reporting for Licensees guidance document, the optional Electricity Reporting Worksheet, and other information on DCC’s website.
