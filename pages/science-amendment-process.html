All active license records listed on your Licenses page now have an Action link for Amendment. This is one route for submitting a Science Amendment. Select the Amendment link to begin.

An alternative method for creating a Science Amendment is to select the License Number to open the record, then select the Create Amendment button.

Amendment Type

On the first page, select Science Amendment and select Continue Application.
